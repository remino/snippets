# Tridactyl for Firefox: Quick Reference

## Modes

- **Normal** ‚Äì default browsing mode
- **Insert** ‚Äì typing in inputs
- **Command** ‚Äì `:`
- **Ignore** ‚Äì Tridactyl disabled on page

## Exit Insert / Command:

- `Esc`
- `Ctrl + [`

## Ignore the next keypress only (send it to the page):

- `Ctrl + v` (normal mode ‚Äúliteral next‚Äù)

## Toggle Ignore mode for the current page:

- `Shift + Escape`
- `Alt + Shift + i`

## Command-line equivalents:

- `:mode ignore`
- `:mode normal`

## Movement

- `j` / `k` ‚Äì scroll down / up
- `h` / `l` ‚Äì left / right
- `gg` ‚Äì top
- `G` ‚Äì bottom
- `d` / `u` ‚Äì half page down / up

History:

- `H` ‚Äì back
- `L` ‚Äì forward

- `f` ‚Äì follow link
- `F` ‚Äì follow link in new tab

- `b` ‚Äì buffer list

- `u` ‚Äì reopen closed tab

- `o` ‚Äì open

- `/` ‚Äì search

- `yy` ‚Äì copy URL
